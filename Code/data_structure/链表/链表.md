# 1️⃣ 链表基础结构

```cpp
struct Listnode {
    int val;
    Listnode* next;
    Listnode(int x) : val(x), next(nullptr) {}
};
```

👉 含义：

- `val`：当前节点存储的值
- `next`：指向下一个节点
- `nullptr`：表示链表结束

------

👉 图示：

```text
[1|next] -> [2|next] -> [3|next] -> nullptr
```

------

# 2️⃣ 链表初始化模板

```cpp
Listnode* init_List(const vector<int>& a) {
    if (a.empty()) return nullptr;

    Listnode* head = new Listnode(a[0]);
    Listnode* cur = head;

    for (int i = 1; i < a.size(); i++) {
        cur->next = new Listnode(a[i]);
        cur = cur->next;
    }

    return head;
}
```

------

👉 图示：

如果数组是：

```text
1 2 3 4
```

初始化后：

```text
head
 ↓
1 -> 2 -> 3 -> 4 -> nullptr
```

------

# 3️⃣ 链表遍历模板

```cpp
Listnode* cur = head;
while (cur) {
    cout << cur->val << " ";
    cur = cur->next;
}
```

------

👉 用途：

- 打印链表
- 遍历查找
- 统计长度

------

👉 图示：

```text
cur
 ↓
1 -> 2 -> 3 -> 4 -> nullptr
```

每次执行：

```cpp
cur = cur->next;
```

就往后走一步。

# 4️⃣ dummy node（虚拟头节点）模板

```cpp
Listnode* dummy = new Listnode(0);
dummy->next = head;
```

------

👉 核心作用：

- 给头节点补一个“前驱”
- 统一头节点和普通节点的处理
- 特别适合删除、插入、拼接

------

👉 图示：

原链表：

```text
1 -> 2 -> 3
```

加 dummy 后：

```text
dummy -> 1 -> 2 -> 3
```

------

👉 什么时候想到 dummy：

- 删除节点
- 插入节点
- 合并链表
- 可能修改头节点

------

# 5️⃣ 删除值为 x 的节点模板

```cpp
Listnode* deleteElement(Listnode* head, int x) {
    Listnode* dummy = new Listnode(0);
    dummy->next = head;
    Listnode* cur = dummy;

    while (cur->next) {
        if (cur->next->val == x) {
            cur->next = cur->next->next;
        } else {
            cur = cur->next;
        }
    }

    return dummy->next;
}
```

------

👉 核心思想：

不是“删除自己”，而是：

```text
前驱 跳过 待删节点
```

------

👉 图示：

删除 2：

```text
cur -> 1 -> 2 -> 3
```

执行：

```cpp
cur->next = cur->next->next;
```

变成：

```text
cur -> 1 ------> 3
```

------

👉 关键点：

- 判断的是 `cur->next`
- 删除后不要急着 `cur = cur->next`
- 否则可能漏删连续节点

------

# 6️⃣ 反转链表模板

```cpp
Listnode* reverseList(Listnode* head) {
    Listnode* pre = nullptr;
    Listnode* cur = head;

    while (cur) {
        Listnode* next = cur->next;
        cur->next = pre;
        pre = cur;
        cur = next;
    }

    return pre;
}
```

------

👉 核心变量：

- `pre`：已经反转好的前半段头节点
- `cur`：当前处理节点
- `next`：提前保存后继，防止断链

------

👉 图示（反转 1 -> 2 -> 3）

初始：

```text
pre   cur
↓     ↓
null  1 -> 2 -> 3
```

第一轮：

```text
next = 2
1->next = null
pre = 1
cur = 2
```

结果：

```text
pre      cur
↓        ↓
1 -> null  2 -> 3
```

继续：

```text
2 -> 1 -> null
3 -> 2 -> 1 -> null
```

------

👉 一句话记忆：

```text
先存 next，再反转指针，再整体后移
```

------

# 7️⃣ 快慢指针模板

```cpp
Listnode* slow = head;
Listnode* fast = head;

while (fast && fast->next) {
    slow = slow->next;
    fast = fast->next->next;
}
```

------

👉 slow 最终位置：

- 中点
- 或偏右中点（视题目而定）

------

👉 图示：

```text
1 -> 2 -> 3 -> 4 -> 5
s
f
```

每轮：

- `slow` 走一步
- `fast` 走两步

最后：

```text
slow 在 3
```

------

👉 用途：

- 找中点
- 判断是否成环
- 找倒数第 k 个节点（变式）

------

# 8️⃣ 判断链表是否有环模板

```cpp
bool hasCycle(Listnode* head) {
    Listnode* slow = head;
    Listnode* fast = head;

    while (fast && fast->next) {
        slow = slow->next;
        fast = fast->next->next;

        if (slow == fast) {
            return true;
        }
    }

    return false;
}
```

------

👉 核心思想：

- 如果有环，快指针迟早会追上慢指针
- 如果无环，快指针会先走到 `nullptr`

------

👉 图示：

```text
1 -> 2 -> 3 -> 4
     ^         |
     |_________|
```

在环里，`fast` 每次比 `slow` 多走一步，所以一定会相遇。

------

# 9️⃣ 找中点模板

```cpp
Listnode* findMiddle(Listnode* head) {

    Listnode* fast = head;

    while (fast && fast->next) {

        fast = fast->next->next;
    }

    return slow;
}
```

------

👉 图示：

```text

```

最终：

```text

```

------

👉 注意：

对于偶数长度链表，返回哪个中点取决于初始化和循环条件。

# 🔟 找倒数第 k 个节点模板

```cpp
Listnode* findKthFromEnd(Listnode* head, int k) {
    Listnode* slow = head;
    Listnode* fast = head;

    for (int i = 0; i < k; i++) {
        if (fast == nullptr) return nullptr;
        fast = fast->next;
    }

    while (fast) {
        slow = slow->next;
        fast = fast->next;
    }

    return slow;
}
```

------

👉 核心思想：

- `fast` 先走 `k` 步
- 然后 `slow` 和 `fast` 同时走
- 当 `fast` 到末尾，`slow` 就在倒数第 `k` 个

------

👉 图示：

链表：

```text
1 -> 2 -> 3 -> 4 -> 5 -> 6 -> 7
```

找倒数第 3 个：

```text
结果是 5
```

因为倒数数：

```text
7(1) 6(2) 5(3)
```

------

# 1️⃣1️⃣ 删除倒数第 k 个节点模板

```cpp
Listnode* deleteKthElementEnd(Listnode* head, int k) {
    Listnode* dummy = new Listnode(0);
    dummy->next = head;

    Listnode* slow = dummy;
    Listnode* fast = dummy;

    for (int i = 0; i < k; i++) {
        if (fast->next == nullptr) return dummy->next;
        fast = fast->next;
    }

    while (fast->next) {
        slow = slow->next;
        fast = fast->next;
    }

    slow->next = slow->next->next;
    return dummy->next;
}
```

------

👉 关键点：

这题不是找“倒数第 k 个节点”，而是找它的**前一个节点**。

所以：

- `slow` 从 `dummy` 出发
- `fast` 先走 `k` 步
- 最后 `slow` 停在待删节点前一个位置

------

👉 图示：

删除倒数第 3 个：

```text
dummy -> 1 -> 2 -> 3 -> 4 -> 5 -> 6 -> 7
```

要删的是：

```text
5
```

最后应该停成：

```text
slow 在 4
```

然后：

```cpp
slow->next = slow->next->next;
```

删掉 `5`

# 1️⃣2️⃣ 合并两个有序链表模板

```cpp
Listnode* mergeTwoLists(Listnode* l1, Listnode* l2) {
    Listnode* dummy = new Listnode(0);
    Listnode* cur = dummy;

    while (l1 && l2) {
        if (l1->val < l2->val) {
            cur->next = l1;
            l1 = l1->next;
        } else {
            cur->next = l2;
            l2 = l2->next;
        }
        cur = cur->next;
    }

    if (l1) cur->next = l1;
    if (l2) cur->next = l2;

    return dummy->next;
}
```

------

👉 图示：

```text
l1: 1 -> 3 -> 5
l2: 2 -> 4 -> 6
```

合并后：

```text
1 -> 2 -> 3 -> 4 -> 5 -> 6
```

------

👉 核心思想：

- 谁小接谁
- `cur` 始终指向结果链表尾部

# 1️⃣3️⃣ 两两交换节点模板

```cpp
Listnode* swapPairs(Listnode* head) {
    Listnode* dummy = new Listnode(0);
    dummy->next = head;

    Listnode* pre = dummy;

    while (pre->next && pre->next->next) {
        Listnode* a = pre->next;
        Listnode* b = a->next;
        Listnode* next = b->next;

        pre->next = b;
        a->next = next;
        b->next = a;

        pre = a;
    }

    return dummy->next;
}
```

------

👉 核心结构：

```text
pre -> a -> b -> next
↓
pre -> b -> a -> next
```

------

👉 图示：

原来：

```text
dummy -> 1 -> 2 -> 3 -> 4
```

交换第一组后：

```text
dummy -> 2 -> 1 -> 3 -> 4
```

再交换第二组：

```text
dummy -> 2 -> 1 -> 4 -> 3
```

------

👉 关键点：

- 必须先保存 `next`
- 必须用 `dummy`
- 最后 `pre = a`

因为交换后：

- `b` 成了前面
- `a` 成了后面

------

# 1️⃣4️⃣ K 个一组反转模板

先准备普通反转函数：

```cpp
Listnode* reverseList(Listnode* head) {
    Listnode* pre = nullptr;
    Listnode* cur = head;

    while (cur) {
        Listnode* next = cur->next;
        cur->next = pre;
        pre = cur;
        cur = next;
    }

    return pre;
}
```

------

主函数：

```cpp
Listnode* reverseKGroup(Listnode* head, int k) {
    Listnode* dummy = new Listnode(0);
    dummy->next = head;
    Listnode* pre = dummy;

    while (true) {
        Listnode* end = pre;

        for (int i = 0; i < k; i++) {
            end = end->next;
            if (end == nullptr) return dummy->next;
        }

        Listnode* start = pre->next;
        Listnode* nextGroup = end->next;

        end->next = nullptr;

        pre->next = reverseList(start);
        start->next = nextGroup;

        pre = start;
    }
}
```

------

👉 核心结构：

```text
pre -> [a b c] -> nextGroup
↓
pre -> [c b a] -> nextGroup
```

------

👉 图示：

链表：

```text
1 -> 2 -> 3 -> 4 -> 5 -> 6 -> 7
k = 3
```

第一组：

```text
1 2 3
```

反转后：

```text
3 2 1
```

整体变成：

```text
3 -> 2 -> 1 -> 4 -> 5 -> 6 -> 7
```

第二组再反转：

```text
3 -> 2 -> 1 -> 6 -> 5 -> 4 -> 7
```

------

👉 关键点：

- `end` 用来找这一组是否够 `k` 个
- 必须先保存 `nextGroup`
- 必须先断开 `end->next = nullptr`
- 反转后：
  - `start` 变尾巴
  - `end` 变头
- 最后 `pre = start`

# 1️⃣6️⃣ 链表题做题思维

看到链表题，先问自己：

------

## 1. 需不需要 dummy？

如果涉及：

- 删除
- 插入
- 头节点可能变化

优先想到 dummy

------

## 2. 是不是快慢指针？

如果涉及：

- 中点
- 环
- 倒数第 k 个

优先想到快慢指针

------

## 3. 是不是反转？

如果涉及：

- 顺序颠倒
- 分组
- 局部调整

优先想到反转

------

## 4. 是不是局部指针重连？

如果涉及：

- 两两交换
- 局部替换
- 复杂拼接

优先画图
