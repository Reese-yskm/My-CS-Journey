# 链表

## 初始化

链表的初始化是链表操作的基础，接下来我将提供几个常见的链表初始化方法。在 C++ 中，链表通常是通过**指针**来实现的，首先你需要定义一个**节点结构体**，然后通过指针来链接各个节点。

### 1. **链表节点结构体的定义**

```cpp
struct ListNode {
    int val;        // 存储的数据
    ListNode* next; // 指向下一个节点的指针

    // 构造函数
    ListNode(int x) : val(x), next(nullptr) {}
};
```

这个结构体 `ListNode` 代表链表中的每个节点，它包含：

- `val`：存储的数据（比如整数 `int`）
- `next`：指向下一个节点的指针

### 2. **初始化一个空链表**

```cpp
ListNode* head = nullptr;
```

这行代码初始化了一个空链表，`head` 是一个指向 `ListNode` 的指针，最初它指向 `nullptr`，表示链表为空。

### 3. **初始化一个单节点链表**

如果你想初始化一个只包含一个节点的链表，可以使用：

```cpp
ListNode* head = new ListNode(10); // 创建一个值为10的节点
```

这会创建一个只有一个节点的链表，`head` 指向这个节点，节点的 `next` 默认为 `nullptr`。

### 4. **初始化一个多节点链表（手动连接）**

```cpp
ListNode* head = new ListNode(1);    // 创建第一个节点
head->next = new ListNode(2);         // 创建第二个节点
head->next->next = new ListNode(3);   // 创建第三个节点
```

这段代码创建了一个链表，包含 3 个节点，链表的内容是：`1 -> 2 -> 3`。每个节点的 `next` 指向下一个节点，最后一个节点的 `next` 默认为 `nullptr`。

### 5. **通过数组初始化链表**

如果你有一个数组，并且想通过这个数组来初始化链表，可以使用循环来实现：

```cpp
int arr[] = {1, 2, 3, 4, 5};
ListNode* head = new ListNode(arr[0]);
ListNode* current = head;

for (int i = 1; i < 5; ++i) {
    current->next = new ListNode(arr[i]);
    current = current->next;
}
```

这段代码将数组 `arr[]` 转换为一个链表，链表的内容是：`1 -> 2 -> 3 -> 4 -> 5`。

### 6. **链表初始化的常见错误**

- **忘记 `next` 指针初始化**：每个新节点的 `next` 都必须初始化为 `nullptr` 或指向下一个节点，否则链表将无法正确工作。
- **内存泄漏**：如果你使用 `new` 动态分配内存，记得在合适的地方使用 `delete` 来释放内存，避免内存泄漏。

### 7. **链表初始化的函数封装（封装常用的初始化操作）**

你也可以把初始化操作封装成函数，这样更方便复用：

```cpp
ListNode* createLinkedList(const std::vector<int>& values) {
    if (values.empty()) return nullptr;

    ListNode* head = new ListNode(values[0]);
    ListNode* current = head;
    
    for (size_t i = 1; i < values.size(); ++i) {
        current->next = new ListNode(values[i]);
        current = current->next;
    }

    return head;
}
```

这个函数接受一个 `std::vector<int>` 作为输入，返回一个包含这些值的链表。你可以像这样调用它：

```cpp
std::vector<int> values = {1, 2, 3, 4, 5};
ListNode* head = createLinkedList(values);
```

## 模版

## 1️⃣ 遍历模板（最基础）

```cpp
ListNode* cur = head;
while (cur != nullptr) {
    // 操作 cur
    cur = cur->next;
}
```

👉 所有链表题的起点

------

## 2️⃣ 反转链表（核心模板）

```cpp
ListNode* reverseList(ListNode* head) {
    ListNode* prev = nullptr; // 初始化前一个节点为 nullptr
    ListNode* curr = head;    // 当前节点指向头节点

    // 遍历链表
    while (curr != nullptr) {
        ListNode* next = curr->next; // 保存当前节点的下一个节点
        curr->next = prev;           // 将当前节点的 next 指向前一个节点
        prev = curr;                 // 将 prev 移动到当前节点
        curr = next;                 // 将 curr 移动到下一个节点
    }

    return prev; // 最终 prev 会指向新的头节点
}
```

👉 必须达到：**默写级别**

------

## 3️⃣ 快慢指针模板

```cpp
ListNode* slow = head;
ListNode* fast = head;

while (fast && fast->next) {
    slow = slow->next;
    fast = fast->next->next;
}
```

👉 slow 最终位置：

- 中点（或偏左中点）

用途：

* 判断是否成环（判断是否slow==fast）
* 找中间节点（最后返回slow）
* 找**倒数**第k个节点（先让fast走k-1步，然后两个指针**同时**走，返回slow，注意不能访问到链表外）

## 4️⃣ dummy node 模板（非常重要）

```cpp
ListNode* dummy = new ListNode(0);
dummy->next = head;

ListNode* cur = dummy;
```

👉 用于：

- 删除节点
- 插入节点
- 合并链表
