# 第1章 图的基本概念

## 1.1 图的定义

图论中的“图”用来描述对象之间的关系。一个图通常记为：

```text
G = (V, E)
```

其中：

- `V` 是顶点集，也叫点集。
- `E` 是边集，用来表示顶点之间的关系。

如果一条边连接顶点 `u` 和顶点 `v`，在无向图中通常记为：

```text
uv 或 {u, v}
```

在有向图中，如果边从 `u` 指向 `v`，通常记为：

```text
(u, v)
```

### 顶点和边

顶点可以理解为研究对象，边可以理解为对象之间的关系。

例如，在“同学认识关系”中：

- 每位同学是一个顶点。
- 两位同学互相认识，就在两个顶点之间连一条边。

### 图的阶和边数

设图 `G = (V, E)`。

- 图的阶是顶点数，记为 `|V|`。
- 图的边数是边的数量，记为 `|E|`。

如果一个图有 `n` 个顶点，就称它为 `n` 阶图。

### 无向图和有向图

无向图中的边没有方向。若 `u` 和 `v` 相邻，那么 `u` 连向 `v` 与 `v` 连向 `u` 是同一件事。

有向图中的边有方向。边 `(u, v)` 表示从 `u` 指向 `v`，一般不同于 `(v, u)`。

### 简单图

简单图通常满足：

- 没有自环，即不存在从某个顶点连回自身的边。
- 没有重边，即任意两个顶点之间最多只有一条边。

对一个 `n` 阶简单无向图，边数最多为：

```text
C(n, 2) = n(n - 1) / 2
```

达到最大边数的简单图叫完全图，记为 `K_n`。

### 度

在无向图中，一个顶点的度是与它关联的边数，记为：

```text
d(v)
```

如果一个顶点没有任何关联边，它的度为 `0`，称为孤立顶点。

### 握手定理

任意无向图中，所有顶点的度数之和等于边数的两倍：

```text
sum d(v) = 2|E|
```

直观理解：每条边有两个端点，所以在统计所有顶点度数时，每条边会被计算两次。

由握手定理可得：

```text
任意图中，奇度顶点的个数一定是偶数。
```

因为所有顶点度数之和是偶数，而偶度顶点的度数和也是偶数，所以奇度顶点的度数和也必须是偶数。若干个奇数相加为偶数，当且仅当奇数的个数为偶数。

### 正则图

如果图中每个顶点的度都等于 `r`，则称这个图为 `r`-正则图。

若一个 `n` 阶图是 `r`-正则图，则所有顶点度数之和为：

```text
nr
```

由握手定理：

```text
2|E| = nr
```

所以边数为：

```text
|E| = nr / 2
```

特别地，若是 `n` 阶 `2`-正则图，则边数为：

```text
|E| = n
```

## 1.2 图的表示

图的表示方式决定了后续算法怎么写。常见表示方法有：

- 边集表示
- 邻接表
- 邻接矩阵

实际写 Python 图论算法时，最常用的是邻接表。

### 边集表示

边集表示直接列出所有边。

例如无向图：

```text
V = {1, 2, 3, 4}
E = {(1, 2), (1, 3), (2, 4)}
```

Python 写法：

```python
vertices = [1, 2, 3, 4]
edges = [(1, 2), (1, 3), (2, 4)]
```

边集表示适合：

- 直接读入题目给出的边。
- Kruskal 最小生成树这类按边处理的算法。
- 判断边数、枚举所有边。

无向图中，`(1, 2)` 和 `(2, 1)` 表示同一条边，通常只存一次。

### 邻接表

邻接表为每个顶点存储它的邻居。

例如：

```text
1: 2, 3
2: 1, 4
3: 1
4: 2
```

Python 可以用字典表示：

```python
graph = {
    1: [2, 3],
    2: [1, 4],
    3: [1],
    4: [2],
}
```

邻接表适合：

- DFS
- BFS
- 连通分量
- 最短路
- 大多数图论算法

无向图添加边时，要给两个方向都添加：

```python
def add_edge(graph, u, v):
    graph.setdefault(u, []).append(v)
    graph.setdefault(v, []).append(u)
```

如果顶点编号是 `0` 到 `n - 1`，也可以用列表表示邻接表：

```python
n = 4
graph = [[] for _ in range(n)]

def add_edge(u, v):
    graph[u].append(v)
    graph[v].append(u)

add_edge(0, 1)
add_edge(0, 2)
add_edge(1, 3)
```

### 有向图的邻接表

有向图只存边的方向。

如果有边 `u -> v`，只需要把 `v` 加入 `u` 的邻接表：

```python
def add_directed_edge(graph, u, v):
    graph.setdefault(u, []).append(v)
    graph.setdefault(v, [])
```

例如：

```python
graph = {
    1: [2, 3],
    2: [4],
    3: [],
    4: [],
}
```

这表示：

```text
1 -> 2
1 -> 3
2 -> 4
```

### 带权图的邻接表

如果边有权值，可以把邻居和权值一起存。

例如边 `1-2` 的权值为 `5`，边 `1-3` 的权值为 `2`：

```python
graph = {
    1: [(2, 5), (3, 2)],
    2: [(1, 5), (4, 7)],
    3: [(1, 2)],
    4: [(2, 7)],
}
```

添加无向带权边：

```python
def add_weighted_edge(graph, u, v, w):
    graph.setdefault(u, []).append((v, w))
    graph.setdefault(v, []).append((u, w))
```

这种表示常用于：

- Dijkstra 最短路
- Prim 最小生成树
- 带权图遍历

### 邻接矩阵

邻接矩阵用一个 `n * n` 的矩阵表示顶点之间是否有边。

若顶点编号为 `0, 1, ..., n - 1`，则：

```text
matrix[i][j] = 1 表示 i 和 j 之间有边
matrix[i][j] = 0 表示 i 和 j 之间没有边
```

Python 写法：

```python
n = 4
matrix = [[0] * n for _ in range(n)]

def add_edge(u, v):
    matrix[u][v] = 1
    matrix[v][u] = 1

add_edge(0, 1)
add_edge(0, 2)
add_edge(1, 3)
```

对于无向图，邻接矩阵是对称的：

```text
matrix[i][j] = matrix[j][i]
```

对于简单图，主对角线通常全为 `0`：

```text
matrix[i][i] = 0
```

### 带权图的邻接矩阵

带权图中，矩阵元素可以存权值。

如果没有边，常用一个很大的数表示无穷大：

```python
INF = float("inf")
n = 4
dist = [[INF] * n for _ in range(n)]

for i in range(n):
    dist[i][i] = 0

def add_weighted_edge(u, v, w):
    dist[u][v] = w
    dist[v][u] = w
```

这种表示常用于 Floyd 算法。

### 三种表示方法的比较

| 表示方法 | 优点 | 缺点 | 常见用途 |
| --- | --- | --- | --- |
| 边集 | 简单，适合按边处理 | 查询邻居不方便 | Kruskal、读入边 |
| 邻接表 | 节省空间，适合遍历 | 查询两点是否相邻较慢 | DFS、BFS、最短路 |
| 邻接矩阵 | 查询两点是否相邻很快 | 空间消耗大 | 稠密图、Floyd |

如果图有 `n` 个顶点、`m` 条边：

- 邻接表空间复杂度约为 `O(n + m)`。
- 邻接矩阵空间复杂度为 `O(n^2)`。

一般来说：

- 稀疏图优先用邻接表。
- 稠密图或需要频繁查询两点是否相邻时，可以用邻接矩阵。

## 本节 Python 小模板

### 从边集建立无向邻接表

```python
def build_undirected_graph(edges):
    graph = {}

    for u, v in edges:
        graph.setdefault(u, []).append(v)
        graph.setdefault(v, []).append(u)

    return graph


edges = [(1, 2), (1, 3), (2, 4)]
graph = build_undirected_graph(edges)
print(graph)
```

### 计算每个顶点的度

```python
def degrees(graph):
    return {v: len(graph[v]) for v in graph}


graph = {
    1: [2, 3],
    2: [1, 4],
    3: [1],
    4: [2],
}

print(degrees(graph))
```

### 计算边数

无向图的邻接表中，每条边会被存两次，所以边数为：

```python
def edge_count(graph):
    return sum(len(neighbors) for neighbors in graph.values()) // 2
```

### 判断握手定理

```python
def check_handshaking(graph):
    degree_sum = sum(len(neighbors) for neighbors in graph.values())
    m = edge_count(graph)
    return degree_sum == 2 * m
```

## 小结

本节重点：

- 图可以记为 `G = (V, E)`。
- 简单图没有自环和重边。
- `n` 阶简单图最多有 `n(n - 1) / 2` 条边。
- 所有顶点度数之和等于边数的两倍。
- 奇度顶点个数一定为偶数。
- `n` 阶 `r`-正则图的边数为 `nr / 2`。
- Python 中最常用邻接表表示图。
- 边集适合按边处理，邻接矩阵适合快速判断两点是否相邻。
